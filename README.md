# 🎯 IELTS Platform

IELTS Platform is an AI-powered preparation platform built for students in Central Asia. It provides mock tests, instant AI feedback on Writing and Speaking, personal study plans, and progress analytics — all in one place, at an affordable price.

---

## ✨ Features

- 🧠 **AI Scoring** — Writing & Speaking feedback
- 📝 **Mock Tests** — All 4 IELTS sections
- 📈 **Progress Tracking** — Band score history
- 🗓 **Personal Study Plan** — Auto-generated daily schedule
- 📚 **Vocabulary** — Spaced repetition system
- 🤖 **Telegram Bot** — Practice via Telegram
- 💳 **Local Payments** — Click & Payme

---

## 🛠 Problems We Solve

- ❌ Expensive foreign platforms — ✅ affordable local pricing
- ❌ No real-time Writing/Speaking feedback — ✅ instant AI evaluation
- ❌ No personal weak area analysis — ✅ smart progress tracking
- ❌ Tools scattered everywhere — ✅ everything in one platform
